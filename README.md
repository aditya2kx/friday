# FRIDAY

FRIDAY is the autonomous dev platform behind the "Jarvis" operator persona. It builds and
operates domain agents (BHAGA, CHITRA, AKSHAYA, CHANAKYA) and evolves itself via a 24/7
fleet. Voice-first input, rich progress surfaces, and a self-instrumenting friction recorder
(R17) make it 1000x faster than manual iteration.

## Setup

```bash
python3 -m venv .venv && .venv/bin/pip install -r requirements.txt && .venv/bin/pytest
```
